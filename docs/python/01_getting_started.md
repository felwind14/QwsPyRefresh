# Getting Started

## Jupyter Notebook Keyboard Shortcuts

-   **Shift-Enter**: run cell, select below
-   **Ctrl-Enter**: : run selected cells
-   **Alt-Enter**: run cell and insert below
-   **Tab**: code completion or indent
-   **Shift-Tab**: tooltip
-   **Matplotlib plotting**: %matplotlib inline
